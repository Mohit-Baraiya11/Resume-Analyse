# Resume-Analyse
🚀 AI Resume ATS Analyzer

An AI-powered Applicant Tracking System (ATS) built using FastAPI, LangChain, and Groq LLM that evaluates resumes against job descriptions and generates a structured analysis including match score, missing skills, improvement suggestions, and professional advice.

📌 Features

📄 Upload resume in PDF format

🧠 Extract technical skills using LLM

📊 Calculate match score (0–1 scale)

❌ Identify missing skills

💡 Provide improvement suggestions

🧑‍💼 Give professional job application advice

🚀 REST API built with FastAPI

📚 Auto API documentation with Swagger

🛠 Tech Stack

Backend: FastAPI

LLM Integration: Groq (LLaMA 3.1)

Framework: LangChain

Validation: Pydantic

PDF Processing: Python PDF utilities

Deployment Ready: Render
